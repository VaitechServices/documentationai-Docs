---
title: Creating a Project
description: How to create a new project in Vaitech Services.
---

## Steps

<Steps>
  <Step title="Open Dashboard">
    Go to `https://dashboard.vaitechservices.com` and login.
  </Step>
  <Step title="Start New Project">
    Click the "Create Project" button in the top right corner.
  </Step>
  <Step title="Fill Details">
    Enter project name and select default settings. Click "Create".
  </Step>
  <Step title="Access Project">
    Your new project appears on the dashboard. Click to open and start managing docs.
  </Step>
</Steps>
