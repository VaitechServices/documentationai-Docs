## Recent Updates

Stay current with the newest releases. Review the highlights below to understand new capabilities, fixes, and changes that enhance your wildlife rescue operations.

<Update label="2024-11-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- **Foliage Mapping**: Interactive maps now let you locate and manage food sources for rehabilitation with GPS integration and customizable markers.
- **Mobile App Push Notifications**: Receive real-time alerts for volunteer shifts, care tasks, and animal health updates directly on your phone.

## Improvements

- Enhanced volunteer roster with drag-and-drop scheduling and conflict detection.
- Faster loading for animal profiles by optimizing database queries.

## Bug Fixes

- Fixed attendance tracking sync issues across devices.
- Resolved PDF report generation errors for financial summaries.

</Update>

<Update label="2024-10-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- **Resource Library Overhaul**: Centralized hub with categorized documents, video embeds, and pinned favorites for quick access.
- **Grant Tracking Dashboard**: Visualize budgets, expenses, and remaining funds with interactive charts.

## Breaking Changes

- Updated API endpoints for animal intake; migrate to `/v2/animals/intake` from legacy paths.
- Volunteer authentication now requires JWT tokens instead of API keys.

## Bug Fixes

- Corrected daily care task automation for recurring medications.
- Improved release location mapping accuracy.

</Update>

<Update label="2024-09-10" description="v1.9.0" tags={["bugfix", "improvement"]}>

## Improvements

- Added completion animations and progress badges for daily care checklists.
- Streamlined financial reports with export options to CSV alongside PDF.

## Bug Fixes

- Fixed no-show alerts not triggering for volunteer shifts.
- Resolved mobile shift management glitches on iOS devices.

</Update>

## How to Update and Stay Informed

Follow these steps to keep your WireCare instance up to date and never miss a release.

<Steps>
  <Step title="Check for Updates" icon="refresh-cw">
    Log in to your dashboard at `https://dashboard.example.com` and navigate to **Settings > Updates**. Click **Check Now** to see available versions.
  </Step>
  <Step title="Apply the Update" icon="upload">
    Select the latest version and follow the guided migration. Backup your data first.
  </Step>
  <Step title="Review Changes" icon="book-open">
    Read the full release notes linked in your update confirmation email.
  </Step>
</Steps>
